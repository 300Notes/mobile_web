# Mobile Web #

### Working locally ###

```bash
$ git clone git@github.com:300Notes/mobile_web.git

$ make deps            # Install application dependencies
$ make minify-sass     # Compile and minify Sass styles
$ make compile-sass    # Compile Sass styles
$ make up              # Start the server
```

### Server requirements ###

```
nodejs  >= 0.8.18
npm     >= 1.2.2
sass    >= 3.2.5
```
